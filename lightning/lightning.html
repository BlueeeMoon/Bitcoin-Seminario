¿Qué es Lightning Network?

Para comprender mejor que es Lightning Network debo explicar cómo funciona Bitcoin.

Bitcoin se ejecuta en la parte superior de un libro de contabilidad llamado blockchain. Debido a que al usar una cadena de bloques para transacciones puede ser costoso, se diseñó Lightning Network.

¿Por qué es costoso hacer transacciones dentro de la cadena?

Conforme se crean más transacciones, se congestiona la red y los usuarios deben pagar tarifas más altas para que su transacción se confirme en el siguiente bloque

Estás transacciones son relativamente lentas, se tardan aproximadamente una hora debido a que cada transacción es vista, validada y almacenada por cada computadora participante, esto genera una gran cantidad de datos y es difícil de escalar. Una vez que los bloques están "llenos", el exceso de transacciones se deja esperar en una cola. Muchos usuarios aumentarán las tarifas que están dispuestos a pagar para comprar espacio para sus transacciones en el siguiente bloque. Además de que las transacciones son públicas.

¿Cómo se resuelve esto?

Para resolver esta congestión de datos se creó Lightning Network que propone una nueva red, una "segunda capa", donde se pueden realizar pagos entre usuarios de igual a igual, sin la necesidad de publicar una transacción en la cadena de bloques de Bitcoin para cada pago. Los usuarios pueden pagarse entre sí en Lightning Network tantas veces como quieran sin incurrir en tarifas en cadena.

Lightning Network es descentralizado también, los pagos se envían a través de una red de canales de micropagos digitales que utilizan contratos inteligentes, conocidos como canales de pago cuya transferencia de valor ocurre fuera de la cadena de bloques. Lightning Network es una red segura de participantes que pueden realizar transacciones a gran volumen y alta velocidad.

¿Cuándo se creó Lightning Network y quién lo creó?

En febrero de 2015, Joseph Poon y Thaddeus Dryja propusieron una posible solución al problema de escalabilidad de Bitcoin, con la publicación de "The Bitcoin Lightning Network: pagos instantáneos escalables fuera de la cadena" y la primera implementación se lanzó en 2018. Desde este momento, Lightning Network brinda a Bitcoin un mundo de posibilidades, incluida la mejora de la privacidad, la velocidad y la escala.

Algunos conceptos básicos que debe comprender:

Nodo: Un nodo es un simplemente una aplicación de software que se ejecuta en un computadora y que participa en la red de Lightning Network.

  • Deben comunicarse de igual a igual con otros nodos Lightning que crean la red.
  • Acceden a la cadena de bloques de Bitcoin para asegurar los fondos utilizados para los pagos.

Algunos conceptos básicos que debes comprender:

Canal: Los canales de pago son el componente principal de Lightning Network y son simplemente una conexión directa entre dos nodos que tienen una relación financiera y que funcionan a través de transacciones de apertura y cierre en cadena.

Enrutamiento: En Lightning Network los pagos se enrutan a través de uno o más canales de pago siguiendo una ruta del remitente al destinatario.

Como mencioné anteriormente para hacer un “pago” con Lihtning Network no necesitas hacerlo dentro de la cadena de bloques, así que los pagos enviados a través de canales de pago se denominan pagos fuera de la cadena de bloques.

¿Cómo se hace una apertura de canal?

El proceso sería el siguiente:

  1. Se realiza una transacción en la cadena de bloques para depositar los fondos en una salida de firma múltiple con la cantidad que desee que contenga el canal y se pagarán tarifas de transacción para abrirlo, a este concepto se le conoce como: transacción de financiamiento.
  2. Una vez que la transacción se transmite en cadena, el canal de pago está abierto y listo para hacer pagos.

¿Por qué usar Lightning Network?

Con Lightning Network puedes hacer pagos instantáneos sin tener que esperar confirmaciones, ya que los fondos se liquidan en segundos, cafés, restaurantes, libros, bares.... además de que se puede utilizarse en terminales de puntos de venta, en transacciones entre dispositivos de usuarios, o en cualquier lugar donde se necesiten pagos rápidos.

Micropagos. Se pueden abrir nuevos mercados con la posibilidad de los micropagos. Lightning permite enviar fondos hasta 0,00000001 de bitcoin sin riesgo de custodia. La blockchain de bitcoin impone actualmente un tamaño mínimo de salida cientos de veces mayor, y una tarifa fija por transacción que hace que los micropagos sean poco prácticos.
